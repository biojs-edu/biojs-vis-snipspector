# snipspector

> Tutorial visualisation component

## Getting Started
Install the module with: `npm install d3 --save`

```javascript
var d3 = require("d3");
d3_show(); // "creates pie chart"
```

## Documentation

You can find documentation [here](http://edu.biojs.net/snipspector_adv/vis_d3/)
